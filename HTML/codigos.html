Para utilizar usamos a tag style e há 3 formas

Existe o CSS inline, ou seja, na mesma linha, onde configuramos apenas uma parte do código

Existe o CSS internal, onde configuramos o style dentro da TAG head e importamos essa configuração.

As importações são feitas pelos atributos ID e Class

O id é usado para apontar uma declaração do CSS, ela deve ser usada apenas 1 vez

O atributo HMTL Class é usado para especificar uma classe para um elemento HTML, ela pode ser utilizar várias vezes

O Css External é uma referencia a um código que é feito a parte e que importamos para o nosso código

Div

A tag Div é o elemento HTML que serve como um contêiner genérico, podendo receber diferentes tipos de contúdos (texto, imagens, etc ...)

Header, section, Article, nav, main e aside

Header é o cabeçalho do nosso site, onde colocamos logo e o menu

Nav é a Tag que representa o nosso menu

Main é oande está o conteudo principal do nosso site, geralmente podemos configurar uma ID aqui

section delimita uma seção de nosso site, tem a mesma dunção da div

Aside é o conteudo lateral em um site

Article são espaços de titulo e texto vinculados ao blog

Todas as tags são pertencentes ao HTML semantico, que é a forma certa de se escrever o conteudo HTML
